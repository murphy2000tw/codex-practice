## 4. Batch 09 實際修改原則
- 不刪除單字。
- 不新增 reading 欄位，正式欄位是 kana。
- 不改變既有 id。

- partOfSpeech 必須與 word、meaning、example 實際用法一致。
- 動詞 meaning 應呈現動作語感，不要寫成名詞解釋。
- 名詞 meaning 應呈現名詞語感，不要寫成動作解釋。
