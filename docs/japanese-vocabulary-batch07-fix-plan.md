## 4. Batch 07 實際修改原則
- 不刪除單字。
- 不改變既有 id。
- 不因表記差異就合併詞條。
- kana 欄位若是純讀音，原則上不得混入漢字。
- 外來語可保留片假名，但需符合自然表記。
- 長音「ー」、促音「ッ」、濁音、半濁音、小ゃゅょ需仔細確認。
- word 與 kana 的表記策略需一致且有理由。
- exampleKana 必須完整對應 example。
- 若 exampleKana 與 kana 讀音不同，需確認是否因語境或詞形變化造成。
- 若只屬於表記策略差異，應優先保留觀察，不要機械式修正。
- 若決定 kana 為純讀音欄，混合詞中的外來語片假名原則上也應轉為對應平假名讀音，並同步修正 exampleKana。
- 若決定 kana 可保留表記資訊，則同一詞根（例如 クリーニング／クリーニング店、センター相關詞）需整批一致。
- 詞形疑義（例如制度名稱）應先人工確認自然度與學習級別，再修改 word、kana、meaning 與例句。
- 只修正 exampleKana 時，也必須確認 exampleMeaning 是否仍對應同一語義。
