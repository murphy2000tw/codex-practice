## 6. 正式匯入前建議

1. 建議人工確認本批 40 個候選詞後再正式匯入，尤其是複合名詞、時間數量詞與「持って来る」表記。
2. 建議由人工審核後，將確定要匯入的 `[ ]` 改成 `[x]`，再交由下一個正式匯入任務讀取；本檔目前不自動勾選任何詞。
3. 最適合優先勾選的詞：`参加`、`修理`、`印刷`、`窓口`、`一緒`、`運動`、`営業時間`、`確認`、`返事`、`料金`、`市役所`、`持ち帰り`。這些詞假名明確、生活情境常見，且閱讀理解用途高。
4. 需要人工再確認的詞：`料理教室`、`クリーニング店`、`ヨガ教室`、`学生料金`、`市立図書館`、`自転車置き場`、`室内用`、`別`、`持って来る`。主要確認點是是否收錄複合詞、詞性標籤與正式表記。
5. 若人工確認完成，下一個正式匯入任務應注意：
   * 只匯入人工已勾選為 `[x]` 的詞，不要匯入本檔暫緩詞。
   * 匯入前再次比對 `vocabulary.json`，避免與 Batch 01 或後續新增詞發生完全重複。
   * 動詞應以辭書形收錄；若收錄 `持って来る`，需先統一表記。
   * 名詞/する動詞可依單字庫格式決定詞性欄與例句，不要同時新增重複的名詞本體與「する」動詞項目。
   * 匯入後應重新執行資料檢查腳本與閱讀單字盤點，並更新匯入紀錄。
