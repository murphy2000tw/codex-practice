- 「お菓子」與「菓子」：可並存。「お菓子」是學習者更常遇到的日常說法；「菓子」較常見於書面、商品分類或複合詞。
- 「お茶」：目前未與「茶」重複；建議保留 N5 常用形式。未來若新增「茶」，需補充「ちゃ / 茶」與「おちゃ / お茶」的語域差異。
- 「お金」：目前未與「金（かね）」重複；建議保留常用形式。未來若新增「金」，需避免與「黃金（きん）」混淆。
- 「弁当」：目前未與「お弁当」重複；建議保留。未來若新增「お弁当」，應說明禮貌 / 常用口語差異。
- 「暖かい」與「温かい」：同讀音但常用語境不同，可保留並補充天氣、物體或情感溫度差異。
- 「早い」與「速い」：同讀音但「早」偏時間早，「速」偏速度快，可保留並強化 meaning 差異。

## 5. 後續建議

- Batch 03 實際修正時，建議先處理「明顯 meaning 錯誤」與「詞性策略不一致」項目，再處理可並存的語域差異項目。
- 對「名詞 + する」詞彙採一致原則：若資料庫同時保留名詞與動詞，名詞項的 partOfSpeech 儘量標為「名詞」，動詞項標為「動詞」，避免第三種「名詞/する動詞」與獨立動詞並存。
- 若「名詞」與「する動詞」分屬不同 level，應逐筆確認是否有教學分級理由；若無理由，建議調整為同級或在報告中記錄保留原因。
