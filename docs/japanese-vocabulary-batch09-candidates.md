## 5. 後續建議

- Batch 09 實際修正時仍應遵守 Batch 03～08 原則：不刪除單字、不新增 reading 欄位、不改 script.js、不因並存詞條就直接合併或刪除既有 id。
- 優先處理「例句只談單字本身」的 する動詞模板，例如「先生は〜という動詞を黒板に書きました」「〜を覚えるために、例文を作りました」「作文で〜を使ってみます」。
- 若 word 是「名詞」但 partOfSpeech 為「名詞/する動詞」，應決定例句要示範名詞用法還是する動詞用法；同一批內需保持一致。
- 若同時存在「名詞/する動詞」與「〜する」動詞，例如 配達/配達する、退院/退院する、残業/残業する，請分別給不同重點：名詞條示範名詞搭配，動詞條示範動作。
- 例句應直接使用目標動詞的核心語義，不要只描述「練習這個詞」「說明這個詞的意思」「把這個動詞寫在黑板上」。
- N5 動詞（練習する、掃除する、洗濯する、仕事する）應比 N4 更生活化、具體化，避免抽象課堂語言學習場景。
- 對「診察する」「試験する」「輸出する」「申し込みする」「会議する」等高風險項目，應先人工確認詞義、詞形與中文語感，再同步修改 meaning、example、exampleKana、exampleMeaning。
- 修正 example 時需同步檢查 exampleKana：ます形、た形、て形、促音、長音、小ゃゅょ、助詞不可遺漏，也不可混入漢字。
- 修正 exampleMeaning 時應翻譯整句日文，不要只翻譯目標單字；若 meaning 有多義，例句翻譯應對應本句使用的那一義。
- 可將 Batch 09 實修分成幾組：高頻生活 N5 動詞、學校/考試動詞、醫療動詞、商務/行政する動詞、移動/旅行動詞，以降低跨領域翻譯不一致。
