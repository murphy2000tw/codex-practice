# 日文外來語 kana / exampleKana 規格確認報告

## 1. 任務目的

本報告只做規格確認，目的是確認目前 `vocabulary.json` 中的 `kana` 與 `exampleKana` 欄位是否允許保留片假名外來語，例如 `センター`、`ホテル`、`レストラン`、`バス`、`テレビ` 等，並釐清 id 3178 `運動センター` 是否符合既有資料慣例。

本任務未修改 `vocabulary.json`、`script.js`、日文功能程式、英文頁面或任何既有資料檔；只新增本報告文件。

## 2. 檢查日期

## 3. 檢查背景

前一個「Batch 03～06 日文單字資料修正任務」將 id 3178 的 `センター` 保留為人工確認項目。原因是 `センター` 屬於外來語片假名，不能只因單一詞條就直接改成平假名，必須先確認整體資料規格與既有慣例。

## 6. kana 欄位目前資料慣例

檢查結果顯示，`kana` 欄位目前實際允許以下型態：

| 型態 | 是否存在 / 是否允許 | 說明 |
| --- | --- | --- |
| 純平假名 | 是 | 大多數漢字詞與和語詞採平假名讀音。 |
| 純片假名 | 是 | 例如 `ポイントカード`、`プール`、`フロント`。 |
| 平假名 + 片假名混用 | 是 | 例如 `わすれものセンター`、`うんどうセンター`、`バスていめい`。 |
| 長音符號 `ー` | 是 | 外來語長音常見於資料中。 |
| 促音 `っ` / `ッ` | 是 | 平假名與片假名促音皆可被資料承載。 |
| 小音 `ゃゅょャュョ` | 是 | 讀音資料中大量存在小音。 |
| 外來語片假名 | 是 | 目前已有多筆外來語保留片假名。 |
| 日文助詞與外來語混合 | 是 | 例如 `うんどうセンター` 與相關例句。 |

判斷：

- 若 `word` 是漢字詞，`kana` 目前以平假名為主。
- 若 `word` 是外來語或含外來語的複合詞，保留片假名較符合現代日文書寫習慣。
- 若 `word` 本身就是片假名外來語，`kana` 可以與 `word` 相同或接近；目前已有 `プール`、`フロント`、`ポイントカード` 等例子。
- 不建議把 `センター` 改成 `せんたー`，因為目前資料已存在 `センター`、`カード`、`バス`、`クリーニング` 等保留片假名的慣例。
- 目前資料已有足夠慣例支持 `kana` 欄位保留片假名外來語。

## 7. exampleKana 欄位目前資料慣例

檢查結果顯示，`exampleKana` 欄位目前實際允許以下型態：

| 型態 | 是否存在 / 是否允許 | 說明 |
| --- | --- | --- |
| 句中漢字轉為平假名 | 是 | 例如 `週末` 轉為 `しゅうまつ`。 |
| 外來語保留片假名 | 是 | 例如 `ホテルのフロントでかぎをもらいます。`。 |
| 平假名與片假名混用 | 是 | 167 筆含片假名或長音符號的 `exampleKana` 都同時含平假名。 |
| 長音符號 `ー` | 是 | 外來語讀音與書寫保留長音符號。 |
| 外來語名詞在假名句中保留片假名 | 是 | 例如 `しゅうまつにプールへいきます。`。 |
| 助詞用平假名，外來語用片假名 | 是 | 例如 `しゅうまつにうんどうセンターでおよぎます。`。 |

判斷：

- `exampleKana` 更接近「讀音輔助句」，不是必須完全平假名化的句子。
- 外來語保留片假名符合日文書寫習慣，也符合目前資料庫中已存在的多筆例句。
- 若 `example` 中有 `センター`，`exampleKana` 保留 `センター` 是合理且符合既有慣例的。
- 不建議把 `センター` 改成 `せんたー`。
