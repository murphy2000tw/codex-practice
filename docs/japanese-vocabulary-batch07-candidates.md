## 5. 後續建議

1. 先明確定義 `kana` 欄位用途：若是「純讀音」，外來語可接受平假名化；若也是「假名表記」，片假名外來語應盡量保留片假名。
2. 對純片假名外來語與漢字/片假名混合詞分開制定規則；不要在同一批同時把所有外來語改為片假名，以免影響既有 ruby rendering 或搜尋行為。
3. 優先修正同詞根不一致項目，例如 `ガス` / `ガス代`、`コピー` / `コピー機`、`クリーニング` / `クリーニング店`。
4. 混合詞若決定採「漢字讀音平假名 + 外來語片假名」策略，需批次整理 `電子レンジ`、`カード払い`、`忘れ物センター`、`運動センター` 等項目。
5. 真正疑似詞形或語義問題（如 `国民保険`、`受付時間`、`机上`）應與 kana/表記修正分開記錄，避免一批修正範圍過大。
6. 實際修正時延續 Batch 03～06 原則：不刪單字、不新增測驗資料、不改 `script.js`；只做能由例句、meaning 與表記策略支持的最小修正。
