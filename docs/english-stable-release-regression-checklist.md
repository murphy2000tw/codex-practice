# 英文網頁穩定版紀錄與回歸檢查清單

## 1. 文件目的

本文件記錄英文學習網站目前已由人工測試確認 OK 的穩定版狀態，並整理日後每次修改前後都可重複使用的回歸測試清單。

本次整理僅建立文件與檢查清單，不新增功能、不修改既有功能邏輯、不改動題庫資料、不改動測驗規則、不改動 UI 樣式。

## 2. 穩定版狀態摘要

目前英文學習網站經過多輪修正後，下列模組與功能已由人工測試確認 OK：

1. 單字一般測驗。
2. 單字聽力測驗。
3. 看圖辨義練習／測驗。
4. 看中文選英文。
5. 填空練習。
6. 填空測驗。
7. 閱讀測驗。
8. 測驗準備頁、作答頁、結果頁狀態切換。
9. 多餘管理按鈕顯示規則。
10. 倒數規則。
11. 答題後正確／錯誤顏色回饋。
12. 聽力結果頁重聽與答案確認。

## 3. 穩定版保護原則

日後修改英文學習網站時，應遵守以下原則，以避免破壞目前已確認 OK 的穩定狀態：

1. 修改前先閱讀本文件，確認本次變更可能影響哪些模組。
2. 修改前記錄目前欲調整的檔案與預期影響範圍。
3. 除非任務明確要求，不改動既有測驗流程、倒數規則、計分規則、結果頁顯示邏輯與 localStorage 紀錄格式。
4. 除非任務明確要求，不改動題庫資料。
5. 除非任務明確要求，不改動 UI 樣式。
6. 修改後依照第 5 節回歸測試清單逐項確認。
7. 若修改涉及共用流程，需至少回測單字一般測驗、單字聽力測驗、填空測驗與閱讀測驗。
8. 若修改涉及作答選項或顏色回饋，需確認未作答前沒有異常顏色，作答後正確／錯誤顏色正常。
9. 若修改涉及結果頁或學習紀錄，需確認 localStorage 中的學習紀錄仍可正確新增、更新與讀取。
10. 每次完成修改後，應在回報中明確說明是否有改動功能邏輯、題庫資料、測驗規則或 UI 樣式。

## 4. 通用回歸檢查項目

以下項目適用於所有英文練習與測驗模組：

- 準備頁是否乾淨，只顯示開始前必要資訊。
- 準備頁是否沒有多餘管理按鈕或不該出現的控制項。
- 進入作答頁後，是否只顯示當前題目需要的內容。
- 未作答前，選項是否沒有異常正確／錯誤顏色。
- 作答後，正確答案與錯誤答案的顏色回饋是否正常。
- 作答後，下一題按鈕或進入下一題流程是否正常。
- 最後一題完成後，是否能正常進入結果頁。
- 結果頁分數、題數、正確／錯誤資訊是否正常。
- 結果頁是否沒有出現作答頁殘留內容。
- localStorage 學習紀錄是否正常新增、更新與讀取。
- 重新整理頁面後，狀態是否符合目前設計，不應出現破版或錯誤狀態。
- 手機版版面是否正常顯示並可操作。
- 桌機版版面是否正常顯示並可操作。

## 5. 模組回歸測試清單

### 5.1 單字一般測驗

- 準備頁是否乾淨，且只顯示開始測驗所需資訊。
- 作答頁是否只顯示單字題目、選項與必要控制項。
- 未作答前，所有選項是否沒有異常正確／錯誤顏色。
- 作答後，正確／錯誤顏色回饋是否正常。
- 選錯時是否能清楚辨識正確答案。
- 下一題流程是否正常，不應跳題或卡住。
- 最後一題後是否正常進入結果頁。
- 結果頁分數、答對題數、答錯題數是否正常。
- localStorage 單字測驗學習紀錄是否正常。
- 手機版與桌機版是否皆能正常作答與查看結果。

### 5.2 單字聽力測驗

- 準備頁是否乾淨，且只顯示開始聽力測驗所需資訊。
- 作答頁是否只顯示聽力題目、播放控制與必要選項。
- 播放或重播語音是否正常。
- 未作答前，所有選項是否沒有異常正確／錯誤顏色。
- 作答後，正確／錯誤顏色回饋是否正常。
- 下一題流程是否正常，語音與選項是否同步切換。
- 最後一題後是否正常進入結果頁。
- 結果頁是否能重聽題目語音。
- 結果頁答案確認是否正常顯示。
- localStorage 聽力測驗學習紀錄是否正常。
- 手機版與桌機版是否皆能正常播放、作答與查看結果。

### 5.3 看圖辨義

- 準備頁是否乾淨，且只顯示看圖辨義練習／測驗所需資訊。
- 作答頁是否正確顯示圖片、題目與選項。
- 圖片載入失敗時是否不影響整體頁面操作。
- 未作答前，所有選項是否沒有異常正確／錯誤顏色。
- 作答後，正確／錯誤顏色回饋是否正常。
- 下一題流程是否正常，圖片與選項是否同步切換。
- 練習模式與測驗模式是否都能正常完成。
- 結果頁是否正常顯示答題結果。
- localStorage 看圖辨義學習紀錄是否正常。
- 手機版與桌機版圖片、選項與結果頁是否正常。

### 5.4 看中文選英文

- 準備頁是否乾淨，且只顯示看中文選英文所需資訊。
- 作答頁是否正確顯示中文提示與英文選項。
- 未作答前，所有英文選項是否沒有異常正確／錯誤顏色。
- 作答後，正確／錯誤顏色回饋是否正常。
- 選錯時是否能清楚辨識正確英文答案。
- 下一題流程是否正常，不應保留上一題選取狀態。
- 最後一題後是否正常進入結果頁。
- 結果頁是否正常顯示答題結果。
- localStorage 看中文選英文學習紀錄是否正常。
- 手機版與桌機版是否皆能正常作答與查看結果。

### 5.5 冠詞練習／測驗

- 準備頁是否乾淨，且只顯示冠詞練習／測驗所需資訊。
- 作答頁是否正確顯示題目句子、空格與冠詞選項。
- 未作答前，冠詞選項是否沒有異常正確／錯誤顏色。
- 作答後，正確／錯誤顏色回饋是否正常。
- 練習模式是否能提供必要回饋且不影響下一題。
- 測驗模式是否依既有規則計分與切換題目。
- 下一題流程是否正常，不應保留上一題顏色或選取狀態。
- 最後一題後是否正常進入結果頁。
- localStorage 冠詞練習／測驗學習紀錄是否正常。
- 手機版與桌機版是否皆能正常作答與查看結果。

### 5.6 閱讀練習／測驗

- 準備頁是否乾淨，且只顯示閱讀練習／測驗所需資訊。
- 作答頁是否正確顯示文章、題目與選項。
- 文章內容、題目與選項是否沒有互相遮擋或錯位。
- 未作答前，所有選項是否沒有異常正確／錯誤顏色。
- 作答後，正確／錯誤顏色回饋是否正常。
- 下一題或下一篇文章流程是否正常。
- 測驗準備頁、作答頁、結果頁狀態切換是否正常。
- 結果頁是否正常顯示閱讀測驗結果。
- localStorage 閱讀練習／測驗學習紀錄是否正常。
- 手機版與桌機版文章閱讀、選項點選與結果頁是否正常。

### 5.7 填空練習／測驗

- 準備頁是否乾淨，且只顯示填空練習／測驗所需資訊。
- 作答頁是否正確顯示句子、空格與選項或輸入區。
- 未作答前，選項或輸入區是否沒有異常正確／錯誤顏色。
- 作答後，正確／錯誤顏色回饋是否正常。
- 練習模式是否能提供必要回饋且不影響下一題。
- 測驗模式是否依既有規則計分與切換題目。
- 下一題流程是否正常，不應保留上一題顏色、答案或選取狀態。
- 最後一題後是否正常進入結果頁。
- 結果頁是否正常顯示填空練習／測驗結果。
- localStorage 填空練習／測驗學習紀錄是否正常。
- 手機版與桌機版是否皆能正常作答與查看結果。

## 6. 修改前檢查流程

每次修改英文學習網站前，建議依序確認：

1. 本次修改是否會碰到英文共用進度、測驗狀態、結果頁或 localStorage。
2. 本次修改是否會碰到題庫資料；若會，需確認任務是否明確允許。
3. 本次修改是否會碰到測驗規則；若會，需確認任務是否明確允許。
4. 本次修改是否會碰到 UI 樣式；若會，需確認任務是否明確允許。
5. 決定本次至少需要回測哪些模組。

## 7. 修改後檢查流程

每次修改英文學習網站後，建議依序確認：

1. 執行與本次修改相關的程式檢查或資料驗證腳本。
2. 依照第 5 節對受影響模組進行人工回歸測試。
3. 若修改共用邏輯，至少回測單字一般測驗、單字聽力測驗、填空測驗與閱讀測驗。
4. 檢查瀏覽器 console 是否沒有新增錯誤。
5. 檢查 localStorage 學習紀錄是否正常。
6. 檢查手機版與桌機版主要流程是否正常。
7. 在交付回報中明確列出已測項目與未測項目。
