## 6. 建議下一步

### A. 安全加入單字庫的詞彙

優先處理建議等級「高」且已有明確假名、詞性與中文意思草案的生活常見詞，例如：`参加`、`修理`。

### B. 需要人工確認後再加入的詞彙

建議等級為「人工確認」的項目應先確認讀音、詞性、是否為專有名詞或是否只是文章標題組合詞，再決定是否收錄。

### C. 不建議加入或暫緩加入的詞彙

人名、單篇標題組合詞、過長片語與只為 ruby 顯示服務的低頻詞可暫緩加入，避免正式單字庫膨脹。

### D. 若要正式執行方案 C，建議採用的資料格式與流程

1. 先以本報告「高」與「中」等級候選詞建立小批次匯入清單。
2. 每筆資料補齊 `level`、`word`、`kana`、`meaning`、`partOfSpeech`、`example`、`exampleKana`、`exampleMeaning`。
3. 動詞一律以辭書形收錄，變化形只放在例句或備註。
4. 匯入前再次檢查同漢字不同假名、同假名不同意思與重複 `word`。
5. 匯入後執行 JSON 驗證、網站健康檢查與 ruby 檢查。
