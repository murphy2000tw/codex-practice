## 4. Batch 08 實際修改原則
- 不刪除單字。
- 不改變既有 id。
- 不因中文翻譯相近就合併詞條。
- meaning 必須清楚，避免只寫「做」「事」「地方」「東西」「情況」「人」「物」等過度模糊說明。
- meaning 應使用台灣學習者容易理解的繁體中文。
- exampleMeaning 必須對應 example，不可多翻或漏翻重要意思。
- exampleMeaning 不要過度直譯，需符合自然中文語感。
- meaning 與 partOfSpeech 必須一致。
- 一詞多義時，應優先保留 N5 / N4 學習者最常見核心用法。
- 若只是翻譯風格差異，沒有明確錯誤，應優先保留觀察。
- 若調整 example，必須同步檢查 exampleKana 是否逐字對應且讀音正確。
